• About

Explicit Check is an application that leverages lyric information from Genius to determine whether a song is clean or explicit.
The default explicit criteria are the Seven Dirty Words, but the user can specify additional words in the format of a
space-separated list (e.g. firstword secondword thirdword). All search fields are case insensitive.

Creds to Github users @farshed (for making a lyrics API based off of the Genius API) and @msmurali
(for building an app that I heavily borrowed from) and Genius for making the original API.
